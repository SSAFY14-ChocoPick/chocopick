# ChocoPick 서비스 기획 요약

## 1. 서비스 개요
ChocoPick은 사용자가 모바일 애플리케이션을 통해 초콜릿 상품을 주문한 뒤,
선택한 오프라인 매장에서 직접 픽업할 수 있도록 지원하는
O2O(Order to Offline) 기반 스마트 픽업 서비스이다.

## 2. 기획 배경
기념일이나 선물 상황에서는 빠른 구매가 중요하지만,
배송 서비스는 시간 지연이 발생할 수 있다.
또한 매장 방문 전에 원하는 상품이 있는지 확인하기 어렵다.

ChocoPick은 이러한 문제를 해결하기 위해
주문과 픽업을 사전에 연결하는 방식을 제안한다.

## 3. 서비스 범위
- 결제 및 정산 기능은 제공하지 않는다.
- 주문 생성, 주문 상태 관리, 픽업 흐름에 집중한다.
- MVP 단계에서는 매장 관리자 전용 화면을 제공하지 않는다.

## 4. 핵심 가치
- 빠른 주문과 단순한 사용자 흐름
- 지도 기반 매장 선택
- 주문 상태 기반 실시간 알림 제공
